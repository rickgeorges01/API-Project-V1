<h1> API Dictionnaire </h1>

<h3>Description</h3> 

Ce projet est une API de traduction qui permet aux utilisateurs de traduire des mots ou des phrases en utilisant divers dictionnaires, tels que le Morse, ASCII, César et binaire. Il est construit avec [FastAPI](https://fastapi.tiangolo.com/) et utilise SQLAlchemy pour la gestion de la base de données.

<h3>Fonctionnalités</h3> 

- Traduction de mots dans différents dictionnaires (traduction mot par mot).
- Support pour plusieurs systèmes de codage et de chiffrement.
- API RESTful pour une intégration facile avec d'autres applications.
